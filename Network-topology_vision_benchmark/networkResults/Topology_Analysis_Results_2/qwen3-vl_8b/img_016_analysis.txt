- **Inadequate Redundancy**:

### 5. Recommendations  
1. **Implement VLAN Segmentation**:

2. **Deploy Router Redundancy via HSRP**:  
   Add a secondary router (e.g., Cisco 2901) connected to Switch0 via a separate link, using **HSRP (Hot Standby Router Protocol)** to ensure continuous gateway access. This eliminates the router as an SPOF and meets high-availability standards for small networks.  

3. **Introduce a Firewall Boundary**:
