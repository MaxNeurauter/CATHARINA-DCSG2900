### 4. Weaknesses & Risks (Cons)  
- **Single Points of Failure (SPOF)**:  
  - The **router (2901)** is a critical SPOF; if it fails, all external connectivity is lost.

- **Lack of Segmentation**: The entire network is a **single broadcast domain** (no VLANs). This:  
  - Exposes all devices to broadcast storms.  
  - Compromises security (e.g., users can directly access each other’s traffic).  
  - Hinders scalability (e.g., 6 PCs here; 50+ would cause performance issues).  
- **Inefficient Layer 2 Design**: The partial mesh with unmanaged STP (indicated by the red dot on the Switch1–2 link) risks **looping** and broadcast storms if STP is misconfigured or disabled.  
- **No Security Boundaries**: There is no firewall or security device between the internal network and the "Subnet," leaving the network vulnerable to external threats.
