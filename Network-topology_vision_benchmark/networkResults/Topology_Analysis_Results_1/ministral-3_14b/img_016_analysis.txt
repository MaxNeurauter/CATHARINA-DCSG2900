### **3. Strengths (Pros)**
✅ **Simplicity** – Easy to deploy and manage for a small network.

### **4. Weaknesses & Risks (Cons)**
❌ **Single Point of Failure (SPOF) – Switch & Router**
   - If **Switch0** fails, all endpoints lose connectivity.
   - If **Router3** fails, all external communication (e.g., internet) is lost.

❌ **No Redundancy**
   - No backup links (e.g., secondary switch, HSRP/VRRP, or link aggregation).
   - No failover mechanisms in case of device or link failure.

❌ **Flat Network (Single Broadcast Domain)**
   - All PCs are in the same collision/broadcast domain, increasing unnecessary traffic.
   - No segmentation for security or performance (e.g., VLANs for different departments).

❌ **No Security Boundaries**
   - No firewalls, ACLs, or segmentation between endpoints.
   - All devices share the same network layer, increasing attack surface.

❌ **No Load Balancing or Scalability**
   - The single switch cannot handle high traffic efficiently without upgrades.
   - Adding more devices may degrade performance.

---

### **5. Recommendations (Industry Best Practices)**
#### **1. Implement VLAN Segmentation & Routing**
   - **Action:** Divide the network into **VLANs** (e.g., VLAN 10 for PCs, VLAN 20 for Servers if added later).
   - **Benefit:**
     - Reduces broadcast traffic.
     - Enables security policies (e.g., restricting traffic between VLANs).
   - **Implementation:**
     - Configure **SVIs (Switch Virtual Interfaces)** on Switch0.
     - Use **Router3** (or a Layer 3 switch if available) for inter-VLAN routing.

#### **2. Introduce Redundancy (High Availability)**
   - **Action:** Add a **second switch** (e.g., Switch1) and configure **Spanning Tree Protocol (STP)** or **Link Aggregation (LACP)**.
   - **Benefit:**
     - Eliminates SPOF by providing backup paths.
     - Improves reliability with **VRRP/HSRP** for the router.
   - **Implementation:**
     - Connect both switches to Router3 and PCs.
     - Enable **STP** to prevent loops.
     - (If budget allows) Deploy **dual routers** with failover.

#### **3. Add Basic Security Controls**

   - **Benefit:**
