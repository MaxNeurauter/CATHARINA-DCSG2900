#### **5. Recommendations (Industry Best Practices)**
To improve this architecture, the following changes should be implemented:

##### **1. Implement a Hierarchical Network Model (Cisco 3-Tier)**
   - **Core Layer:** High-speed backbone (e.g., redundant switches/routers).

   - **Access Layer:** Endpoint connections (segmented via VLANs).
   - **Example:**
     - Use **two distribution switches** for redundancy.
     - Assign **VLANs** (e.g., VLAN 10 for HR, VLAN 20 for Finance, VLAN 30 for IoT).

##### **2. Introduce Redundancy & High Availability (HA)**
