### **3. Attack Path**
#### **A. Kerberoasting / AS-REP Roasting**
- **Authenticated Users** having **Replicating Directory Changes** and **password-related permissions** allows attackers to:
  - Extract Service Principal Names (SPNs) for **Kerberoasting**.
  - Force AS-REP requests for **AS-REP Roasting** (if accounts lack pre-authentication).

#### **B. Privilege Escalation (Golden Ticket / Silver Ticket)**
- **ENTERPRISE DOMAIN CONTROLLERS** having **Replication permissions** allows an attacker to:
  - Modify AD replication data (e.g., inject a **Golden Ticket**).
  - Create **Silver Tickets** for service impersonation.

#### **C. Lateral Movement & Persistence**
- **ANONYMOUS LOGON Read access** → Enables **LDAP enumeration** (e.g., BloodHound recon).
- **Creator Owner writes** → Allows attackers to **plant malicious objects** (e.g., scheduled tasks, backdoor accounts).

#### **D. Pass-the-Hash / Pass-the-Ticket Attacks**
- **Authenticated Users** having **Update password not required bit** → Allows **pass-the-hash** attacks if credentials are leaked.
