*   **Privilege Escalation:**  The "Enterprise Admins" group membership is extremely dangerous.  If an attacker gains access to this account, they have essentially complete control over the Active Directory domain. This allows them to create new accounts, modify existing user and group permissions, and generally wreak havoc on the entire infrastructure.  The "Password never expires" is a massive compounding factor here; it means the attacker doesn't have to worry about the credential expiring.

**4. Remediation:**

Here's a step-by-step remediation plan:

2.  **Enable Password Expiration:**  Uncheck the "Password never expires" option. Configure a reasonable password expiration policy (e.g., 90 days with a grace period).
