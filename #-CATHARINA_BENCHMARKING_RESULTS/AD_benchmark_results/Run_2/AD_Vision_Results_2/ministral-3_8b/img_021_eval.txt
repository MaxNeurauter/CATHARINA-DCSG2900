- **Impact**: Compromised credentials for high-privilege users (e.g., `Domain Admins`).

#### **B. Credential Theft via Hardcoded Password**
- **Attack Vector**: The password **"Heartsbane"** for `samwell.tarly` is stored in plaintext. If an attacker gains access to this screenshot or metadata, they can directly log in as `samwell.tarly`.

#### **C. Privilege Escalation via Domain Admin Access**
