The "Everyone" group having "Change password" on a user object is a misconfiguration. In Active Directory, the "Change password" permission for user objects should only be granted to authorized entities (e.g., "Users" group, "Authenticated Users" group). Granting it to "Everyone" allows any user (including anonymous users, guests, or unauthorized actors) to reset the password, enabling account takeover.

### Assessment  
**Misconfigured**  

### Attack Path  
**Unauthorized Password Change Attack**  
- **Technical Impact**: An attacker can reset the password of `robby.stark` without knowing the original password. This allows them to authenticate to the domain using the new password, leading to full access to the user's resources and potentially escalating privileges (e.g., to the domain's password policies or other objects).  

### Remediation  
1. **Open Advanced Security Settings**: In Active Directory Users and Computers, right-click `robby.stark` → *Properties* → *Security* tab → *Advanced* → *Permissions* (or open via PowerShell with `Get-ACL -Path "AD:\CN=robby.stark,OU=Users,DC=robby,DC=stark"`).  
2. **Remove "Everyone" Permission**: In the *Permission Entries* list, select the entry for **"Everyone"** → click **"Remove"**.  
3. **Reapply Restricted Permissions**:  
   - Ensure **"Users"** (or **"Authenticated Users"**) has the **"Change password"** permission (this is standard for user objects).  
   - Verify **"Domain Admins"**, **"Enterprise Admins"**, and **"Administrators"** groups have **"Special"** access (normal for admin objects).  
4. **Confirm Changes**: Click **"Apply"** → **"OK"**.  
5. **Validate**: Verify the ACL no longer includes "Everyone" with "Change password" (e.g., via `Get-Acl` or the GUI).  

### Why This Fix Works  
By restricting "Change password" to only authorized groups (e.g., "Users"), the configuration aligns with best practices. This prevents unauthorized actors from resetting passwords, mitigating account takeover risks. The fix does not require additional tools—it is a standard ACL adjustment.
