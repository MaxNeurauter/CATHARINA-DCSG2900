  2. Discovers `jon.snow` has an SPN (`CIFS/thewall...`).
  3. Requests a **Kerberos Ticket Granting Ticket (TGT)** for the account (using `Rubeus` or `Impacket`).
  4. Cracks the **TGS (Ticket Granting Service) hash offline** (since Kerberos uses RC4 or AES encryption).
- **Impact:**
  - If the password is weak (e.g., "WinterIsComing123"), the attacker gains **full access to the user’s session**.

- **Impact:**
